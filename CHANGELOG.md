# Changelog

All notable changes to PolyWorld will be documented in this file.

## [1.0.0] - 2025-03-17

### Added
- 908 real-world cities across all 7 continents
- Interactive Leaflet map with CartoDB Dark theme
- Phantom wallet integration for Solana
- Three-tier rarity system (Epic, Rare, Common)
- Fixed-size hexagon markers that don't scale with zoom
- Continental filtering in sidebar
- Buy/sell land mechanics with protocol liquidity
- Daily yield calculations (2-4% APY based on rarity)

### Technical
- React 18 frontend with Babel standalone
- Responsive design for desktop
- External cities data file for performance
- Modular component architecture

## [0.9.0] - 2025-03-13

### Added
- Initial prototype with 200 cities
- Basic map visualization
- Wallet connection framework
- Land purchase modal

### Changed
- Migrated from inline city data to external JSON
- Improved marker styling

## [0.1.0] - 2025-03-12

### Added
- Project initialization
- Basic HTML structure
- Leaflet map integration
- Concept documentation
